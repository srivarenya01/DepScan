# Security Policy

## Supported Versions

Currently, DepScan is a **Work In Progress** and does not have formal versioned releases. All security fixes will be applied directly to the `main` branch. 

## Reporting a Vulnerability

Security is a priority. If you discover any security vulnerabilities, please do not disclose them publicly via GitHub issues. 

Instead, report them privately by emailing: **srivarenya.mudumba@gmail.com**

Please include the following in your report:
* A description of the vulnerability.
* Steps to reproduce the issue.
* (Optional) Any potential mitigation or patch you have identified.

You should receive a response acknowledging receipt within 48 hours. Subsequent updates will be provided as the vulnerability is triaged, patched, and publicly disclosed.
